# О сервере

## ▌ Особенности сервера

> * **Большой мир без вайпов**
> * **Гриферство запрещено в любых его проявлениях**
> * **Новые крафты**
> * **Аукцион и личные магазины**
> * **Минимальные цены для аукциона**
> * **Рисование пиксель-артов**
> * **Прорисовка 8 чанков**
> * **Выключена крыша ада**
> * **Титулы и значки**
> * **Создание гильдий и заключение браков**
> * **Ежедневные награды**

## ▌ Игровые чаты

> Для удобства мы создали несколько игровых чатов:

> ▪ **L** — **Локальный**
> Сообщения видны в радиусе **100** блоков.
>
> ▪ <span style="color: #3b82f6">**G**</span> — **Глобальный**
> Сообщения видны всем во всех мирах.
> Нужно написать знак **«!»** перед текстом.
>
> ▪ <span style="color: #22c55e">**T**</span> — **Торговый**
> Виден везде, но используется только для продажи ресурсов.
> Нужно написать знак **«$»** перед текстом.

## ▌ Доступные команды

::: tip Совет
Полный список команд всегда доступен в игре через **/help**.
:::

> **Основные:**
> *   `/rtp` — Рандомная телепортация
> *   `/spawn` — Телепорт на спавн
> *   `/menu` — Меню сервера
> *   `/donate` — Привилегии сервера
> *   `/kits` — Комплекты (киты)
> *   `/hdb` — Головы для декораций
> *   `/art` — Рисование

> **Дом и регионы:**
> *   `/sethome` — Установить точку дома
> *   `/homes` — Меню с домами
> *   `/ps help` — Помощь по привату территорий
> *   `/ps get` — Получить защитный блок

> **Экономика:**
> *   `/money` — Баланс монет
> *   `/pex` — Баланс пексов (донат-валюта)
> *   `/pay <ник> <сумма>` — Отправить игроку деньги
> *   `/baltop` — Топ игроков по балансу
> *   `/ah` — Аукцион

> **Взаимодействие:**
> *   `/msg <ник>` — Личное сообщение
> *   `/r` — Ответ на последнее сообщение
> *   `/tpa <ник>` — Запрос на телепортацию к игроку
> *   `/ignore <ник>` — Игнорировать игрока
> *   `/ignorelist` — Список игнорируемых игроков
> *   `/discord` — Дискорд сервера

> **Эмоции и действия:**
> *   `/sit` — Сесть на любой блок
> *   `/lay` — Лечь на любой блок
> *   `/crawl` — Ползти
> *   `/spin` — Крутиться
> *   `/premium` — Отметить аккаунт как лицензионный
